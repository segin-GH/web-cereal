# OATS

### INTRODUCTION

A light-weight Serial agent for web-cereal

#### Getting Started

1. install Oats 
```bash
wget https://github.com/segin-GH/web-cereal/raw/main/oats/oats-agent/dist/oats-agent/oats-agent
```
2. Update permesion
```
chmod +x ./oats-agent/oats-agent
```
  
4. Run Oats.
```
./oats-agent/oats-agent
```
